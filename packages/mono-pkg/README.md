## Usage

### Install

```sh
# npm
npm i mono-pkg

# yarn
yarn add mono-pkg

# pnpm
pnpm add mono-pkg

# bun
bun add mono-pkg
```

### API

```js
import echo from 'mono-pkg'

echo()
```
